リポジトリオーナー
 ✓ ✓ ✓ ✓ GitHub アカウント + PAT
🔧 編集者
Collaborator 権限
 ✓ ✓ Git 知識が必要 — Collaborator招待 + 個人PAT
👤 閲覧者
一般社員・URL共有のみ
 ✓ — — — URLを知るだけでOK
1
リポジトリに招待
GitHub → SYL-member/internal-dashboard → Settings → Collaborators → 相手のGitHubアカウント名かメアドで招待 → 相手がメールで承諾
2
相手が PAT を発行
GitHub にログイン → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate → repo スコープにチェック → 発行（一度しか表示されない）
3
ダッシュボードで使えるようになる
DBページを開く → 「☁️ GitHub に保存」をクリック → 初回のみ PAT を入力 → 以降は自動保存 → データ更新後に保存で本番反映
