🔒 アクセス制御（現状）
Pages URL は誰でも閲覧可能（URLを知っていれば）。
現状は機密保護なし。

Mgr用ページを守るには：
→ GitHub Pages を Private にする（有料プラン）
→ または Basic 認証をかける（要サーバー）
→ または URL を社内のみ共有（運用で管理）
👥 社内展開の手順
閲覧者を増やす：URL を共有するだけ

編集者を増やす：
1. GitHub → Settings → Collaborators → Invite
2. 招待者が PAT を発行（repo スコープ）
3. 「GitHubに保存」ボタンで保存可能に

新しいDB追加：
既存 HTML をテンプレートにコピー → Claude で改修
