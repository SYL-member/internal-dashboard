2
データ更新フロー（毎月の作業）
1
Excel を取得 SharePoint
SharePoint から当月の請求書・事業別収支 Excel をダウンロードする。
2
DBページでアップロード ブラウザ操作
各ダッシュボードの「📊 Excelで更新」ボタンをクリックし、ダウンロードしたファイルを選択する。
3
自動解析・グラフ更新 XLSX.js / Chart.js
XLSX.js がブラウザ内でシートを解析し、グラフ・テーブルが自動的に再描画される。サーバー不要・インターネット不要（CDN読込済の場合）
4
追加入力（必要な場合のみ） 手動
コメント・前年比較値・目標値など、Excelから自動取得できない項目を編集パネルから手入力する。
5
「GitHub に保存」をクリック PAT 認証
保存ボタンを押すと GitHub REST API 経由でファイルが push される。初回のみ PAT を入力（以降は自動）。
✓
1〜2分で全社員に反映 GitHub Pages 自動ビルド
GitHub Actions が push を検知してビルドを実行。約1〜2分後に本番 URL が更新され、全社員が最新データを閲覧できる。

Settings → Developer settings →
Personal access tokens → Tokens (classic) →
Generate new token → repo にチェックして発行
