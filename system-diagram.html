📋 編集者を追加する手順（Collaborator招待）
1
リポジトリに招待
GitHub → Kei-SYL/internal-dashboard
→ Settings → Collaborators
→ 相手のGitHubアカウント名 or メアドで招待
→ 相手がメールで承諾
2
相手が PAT を発行
相手が GitHub にログイン
→ Settings → Developer settings
→ Personal access tokens → Tokens (classic)
→ Generate new token → repo にチェック → 発行
→ トークンをどこかにメモ（一度しか表示されない）
3
ダッシュボードで使えるようになる
DBページを開いて「☁️ GitHubに保存」をクリック
→ PAT 入力を求められる（初回のみ）
→ 以降は自動入力される
→ データ更新後に「保存」ボタンで反映
